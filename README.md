📁 CODSOFT Python GUI Projects Collection

This repository contains a collection of beginner-friendly Python GUI-based applications developed as part of an internship. Each project demonstrates core programming concepts along with user-friendly graphical interfaces.

🚀 Projects Included
🧮 Calculator Project
File: calculator.py
A GUI-based calculator that performs:
Addition
Subtraction
Multiplication
Division
Designed with interactive buttons for easy user input.

📇 Contact Book Project
File: contact_book.py
A graphical contact management system that allows users to:
Add new contacts
View contact list
Search contacts
Delete contacts
Provides a simple and clean interface for managing data.

🔐 Password Generator Project
File: password_generator.py
A GUI tool that:
Generates strong and secure passwords
Allows users to select password length
Ensures better security with randomness

✊ Rock-Paper-Scissors Game
File: rock_paper_scissors.py
A GUI-based interactive game where:
Users play against the computer
Buttons are used for selections
Instant results are displayed on screen

📝 To-Do List Project
File: todo.py
A simple task manager with GUI features:
Add tasks
View tasks
Delete completed tasks
Helps in organizing daily activities effectively.
🛠️ Technologies Used
Python 3
GUI Library (Tkinter / Custom GUI framework used in Python)
📌 How to Run

Clone the repository:

git clone <https://github.com/manaviha/Codsoft.git>

Navigate to the project folder:

cd CODSOFT

Run any project:

python filename.py
